## Cara Menjalankan Notebook

Secara lokal:

1. Pastikan Python dan library yang dibutuhkan sudah terpasang.
2. Jalankan Jupyter Notebook atau Jupyter Lab pada folder repository ini.
3. Buka file `.ipynb` yang diinginkan lalu jalankan sel dari atas ke bawah.

Dengan Google Colab:

1. Unggah file notebook ke Google Colab.
2. Jalankan `Runtime -> Run all`.
3. Untuk Pertemuan 3, pastikan file `housing_dirty.csv` juga tersedia.
